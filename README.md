# Random Password Generator 隨機密碼生成器

## 介紹

隨機生成密碼

## 功能

- 選擇密碼長度
- 選擇密碼組格式
- 可自行輸入不要的密碼字元

## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

```bash
npm install
```

4. 安裝完畢後輸入:

```bash
npm run start
```

5. 若看見此行訊息表示順利運行，打開瀏覽器輸入以下網址

```bash
App is running on http://localhost:3000
```

6. 若暫停輸入:

```bash
ctrl + c
```

## 開發工具

- Node.js 18.17.1
- Express 4.18.2
- Express-Handlebars 7.1.2
